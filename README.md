## React Theming

A small Gatsby site to learn CSS-in-JS with `styled-components`.

[Demo Link](https://react-theming.now.sh/)

### Installation

```
$ git clone https://github.com/wlto/react-theming.git
$ cd react-theming/
$ npm install
```

### Running development server

```
$ npm start
```

### Build for production

```
$ npm run build
```

### Test production

```
$ npm run serve
```
